# 音频资源目录

默认配置中的所有报时音频路径均指向本目录（相对路径），例如：

- `music.mp3` —— 报时前奏音乐
- `bell.mp3` —— 钟声（整点重复 N 次）
- `hour_0.mp3` ~ `hour_23.mp3` —— 整点语音（“X 点”）
- `half_0.mp3` ~ `half_23.mp3` —— 半点语音（“X 点半”）

请在部署时把对应的 `.mp3` / `.wav` 文件放入此目录。

> 说明：应用的实际运行时配置保存在 `%AppData%\ClockSystem.WinUI\clock.json`，
> 首次启动时若不存在会自动生成默认值。仓库根目录的 `clock.json` 仅作模板示例。
> 也可在“设置 → 文件路径”中改用绝对路径指向任意位置的音频文件。
